# BNF SMP Website

Official website for the **Barney & Friends SMP** Minecraft server.

🌍 Live site: https://bnfsmp.duckdns.org/

---

## About

This is a lightweight, self-hosted website for a whitelist-only Minecraft SMP server.

It provides:
- Server IP display
- Copy-to-clipboard button
- Real-time server status (online/offline)
- Player count
- Discord invite access

---

## ⚙️ Features

- 🟢 Live server status using mcsrvstat API  
- 📋 One-click IP copy  
- 🎮 Minecraft-style UI and font  
- 📱 Fully responsive design (mobile + desktop)  
- ⚡ Cached API requests for performance  


---

## 🔗 API Used

- https://api.mcsrvstat.us/

Used to fetch:
- Server status
- Player count
